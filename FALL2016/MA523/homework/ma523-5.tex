  is, if \(O\) is an orthogonal \(n\times n\) matrix and we define
  \(v(x)\defeq u(Ox)\), \(x\in\bbR^n\), then \(\Delta v=0\).
\end{problem}
\begin{solution}

  \[
    O=\begin{bmatrix}

    \end{bmatrix}
  \]
  and set \(y(x)=Ox\). We will show that
  \[
    \Delta v(y)=0
  \]
  where \(v(x)=u\bigl(y(x)\bigr)\).

    Dy
    &=D(Ox)\\
